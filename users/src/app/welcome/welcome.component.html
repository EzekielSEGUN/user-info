Welcome to User Information Display App.

This is a simple application for fetching user's information using http mock data by calling the get http request.

To add a new use click "Add User" Fill the forms with the corresponding data and click submit

To view if the user information is added click on "User List" to see the full information of users

To view just first name click on "First Name"

To view just last name click on "Last Name"

To view just gender click on "Gender"

To view just birthday click on "Birthday"
